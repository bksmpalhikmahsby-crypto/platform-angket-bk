Integrasi Google Sheets
Belum terhubung
Setelah terhubung, setiap hasil angket akan otomatis tersimpan ke Google Sheets Guru BK secara real-time.
📌 Tujuan: Setiap siswa mengisi angket → hasil langsung otomatis masuk ke Google Sheets Guru BK secara real-time. Guru BK bisa memantau, menganalisis, dan mengunduh data kapan saja.

LANGKAH 1 — Buat Google Spreadsheet

  1. Buka sheets.google.com (login dengan akun Google sekolah)
  2. Klik tombol "+" atau "Blank spreadsheet"
  3. Beri nama: "Data Angket BK SMP Al Hikmah Surabaya"
  4. Spreadsheet siap — nanti akan otomatis terbuat sheet terpisah untuk setiap jenis angket

LANGKAH 2 — Buka Google Apps Script

  1. Di Spreadsheet, klik menu Extensions (Ekstensi) → Apps Script
  2. Akan terbuka tab baru editor kode
  3. Hapus semua kode yang sudah ada di sana (biasanya function myFunction(){})
  4. Salin dan tempel kode di bawah ini:
function doPost(e) { try { var data = JSON.parse(e.postData.contents); var ss = SpreadsheetApp.getActiveSpreadsheet(); var sheetName = data.angket || 'Data BK'; var sheet = ss.getSheetByName(sheetName); if (!sheet) { sheet = ss.insertSheet(sheetName); } if (sheet.getLastRow() === 0) { sheet.appendRow(Object.keys(data)); } sheet.appendRow(Object.keys(data).map(function(k){ return data[k] || ''; })); return ContentService .createTextOutput(JSON.stringify({status:'ok'})) .setMimeType(ContentService.MimeType.JSON); } catch(err) { return ContentService .createTextOutput(JSON.stringify({status:'error',msg:err.toString()})) .setMimeType(ContentService.MimeType.JSON); } } // Fungsi untuk test manual dari spreadsheet function testDoPost() { var testData = { timestamp: new Date().toISOString(), nama: 'Siswa Test', absen: '1', kelas: 'VII-A', jk: 'Laki-laki', cita: 'Dokter', angket: 'TEST' }; var e = { postData: { contents: JSON.stringify(testData) } }; var result = doPost(e); Logger.log(result.getContent()); }

LANGKAH 3 — Simpan dan Deploy sebagai Web App

  1. Klik ikon 💾 Save (atau Ctrl+S) di Apps Script Editor
  2. Klik tombol biru "Deploy" di kanan atas → pilih "New deployment"
  3. Klik ikon ⚙️ di sebelah "Select type" → pilih "Web app"
  4. Isi deskripsi (opsional): "Angket BK SMP Al Hikmah"
  5. Bagian "Execute as" → pilih "Me (nama emailmu)"
  6. Bagian "Who has access" → pilih "Anyone" ⚠️ (penting! agar siswa bisa kirim data)
  7. Klik "Deploy"
  8. Muncul popup "Authorization required" → klik "Authorize access"
  9. Pilih akun Google → klik "Advanced" → "Go to ... (unsafe)" → "Allow"
  10. Setelah berhasil, muncul Web app URL — salin URL tersebut!
  11. Tempel URL di tab "Hubungkan URL" di platform ini
⚠️ Catatan Penting:
• URL Web App hanya berlaku selama deployment aktif. Jika kamu membuat deployment baru, URL akan berubah.
• Jika ada perubahan kode, klik Deploy → "Manage deployments" → klik ✏️ edit → ubah versi → Save.
• Jika muncul error "403 Forbidden", pastikan "Who has access" sudah diset ke "Anyone".
• Data yang masuk bisa langsung dilihat di Google Sheets realtime.

LANGKAH 4 — Verifikasi Data di Sheets

Setelah siswa mengisi angket dan data terkirim, buka Google Sheets. Akan muncul sheet baru sesuai nama angket (misal: "Angket Gaya Belajar"). Setiap baris adalah data satu siswa dengan kolom: timestamp, nama, kelas, absen, dan skor tiap dimensi.

🌐 Tujuan: Platform angket ini bisa diakses siswa dari HP/laptop masing-masing melalui alamat web (URL), tanpa perlu download file HTML. Berikut beberapa pilihan hosting GRATIS.
🥇 PILIHAN TERBAIK — GitHub Pages (Gratis, Stabil, Profesional)
Kelebihan: Gratis selamanya, URL profesional (namamu.github.io/angket-bk), tidak ada iklan, cepat, bisa custom domain.
URL contoh: https://bkalhikmah.github.io/angket/

Cara Setup:
  1. Buka github.com → klik "Sign up" → buat akun gratis
  2. Setelah login, klik tombol "+" di kanan atas → "New repository"
  3. Isi nama repository: angket-bk (tanpa spasi)
  4. Pilih "Public" → centang "Add a README file" → klik "Create repository"
  5. Klik "Add file" → "Upload files"
  6. Upload file angket_bk_alhikmah.html yang sudah kamu download
  7. Rename file menjadi index.html (klik nama file → edit → ganti nama → commit)
  8. Klik menu "Settings" → scroll ke bawah → cari bagian "Pages"
  9. Di bagian "Source" → pilih branch: "main" → folder: "/(root)" → klik "Save"
  10. Tunggu 1-3 menit → akan muncul URL: https://[username].github.io/angket-bk/
  11. Bagikan URL tersebut ke seluruh siswa! ✅
🥈 ALTERNATIF MUDAH — Netlify Drop (Drag & Drop, Langsung Jadi!)
Kelebihan: Paling mudah — tinggal drag & drop file, langsung dapat URL dalam 30 detik. Gratis, tanpa daftar pun bisa.
URL contoh: https://amazing-goldfish-abc123.netlify.app

Cara Setup (super mudah!):
  1. Buka app.netlify.com/drop
  2. Buat folder baru di komputermu, beri nama angket-bk
  3. Masukkan file HTML ke dalam folder tersebut, rename menjadi index.html
  4. Drag & drop folder tersebut ke kotak besar di halaman Netlify Drop
  5. Tunggu upload selesai (10-30 detik)
  6. Langsung dapat URL yang bisa diakses siapa saja! Salin dan bagikan ke siswa ✅
  7. Opsional: Daftar akun Netlify gratis untuk menyimpan URL permanen dan bisa update file kapan saja
🥉 PILIHAN SEKOLAH — Google Sites (Terintegrasi Akun Google)
Kelebihan: Terintegrasi dengan akun Google sekolah, mudah dikelola, bisa diproteksi hanya untuk domain sekolah.
URL contoh: https://sites.google.com/view/angket-bk-alhikmah

Cara Setup:
  1. Buka sites.google.com → login dengan akun Google sekolah
  2. Klik "+" untuk membuat site baru
  3. Beri nama: "Angket BK SMP Al Hikmah Surabaya"
  4. Di editor, klik "Insert" → "Embed" → "Embed code"
  5. Salin seluruh isi file HTML dan tempel di kotak embed
  6. Klik "Publish" → atur URL yang mudah diingat
  7. Bagikan URL ke siswa ✅
⚠️ Catatan: Google Sites menggunakan iframe untuk embed, yang mungkin membatasi beberapa fitur JavaScript. Pilihan GitHub Pages atau Netlify lebih direkomendasikan untuk fungsi penuh.
📱 TIPS DISTRIBUSI — QR Code untuk Siswa
Setelah punya URL platform, buat QR Code agar siswa tinggal scan dengan HP:
  1. Buka qr.io atau qr-code-generator.com
  2. Masukkan URL platform angket BK-mu
  3. Download QR Code dalam format PNG/SVG
  4. Print dan tempel di papan pengumuman, atau kirim via WhatsApp Group kelas
  5. Siswa tinggal scan → langsung terbuka platform angket di HP mereka ✅
💡 Tips: Jika sekolah punya domain (.sch.id), tanyakan kepada operator IT sekolah untuk membuat subdomain seperti bk.alhikmah.sch.id yang mengarah ke platform ini. Jauh lebih profesional!
